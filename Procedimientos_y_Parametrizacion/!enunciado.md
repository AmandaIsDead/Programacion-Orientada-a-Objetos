# Tarea - Procedimientos y Parametrización
El presente ejercicio tiene como objetivo la transición de una estructura de código lineal y estática hacia un modelo basado en la modularización y la abstracción. 
Se busca que comprendas la utilidad de losprocedimientos para encapsular lógica y permitir la reutilización de código mediante el uso de parámetros.
Objetivos del Ejercicio: 
- Refactorización: Adaptar la composición gráfica realizada en el primer ejercicio para que su lógica resida dentro de un nuevo procedimiento propio.
- Parametrización: El procedimiento debe ser capaz de recibir variables externas que modifiquen su ubicación y escala en el lienzo.
- Implementación de Interactividad: Vincular la ejecución del procedimiento a eventos de entrada (mouse) para generar una respuesta dinámica en el programa.
