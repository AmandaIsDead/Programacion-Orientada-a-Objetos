# Tarea - Arrays 2D (Grilla Interactiva)
Crear un programa en Processing que dibuje una grilla de celdas.El programa debe "recordar" qué celdas han sido activadas por el usuario mediante un clic,
cambiando su apariencia visual (entre un color sólido o un diseño/estampa personalizada) según el valor almacenado en una matriz.
